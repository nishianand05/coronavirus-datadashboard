Coronavirus Data Dashboard

About

Hello! I am Nishi Anand, a B.Tech Computer Science and Engineering student from SRM Institute of Science and Technology, Tamil Nadu, India. This is a data visualization dashboard to visualize data related to the coronavirus pandemic. I have created a Map visualization, Bar chart, Pie chart and a Scatterplot to visualize mainly 3 attributes - Number of cases, deaths and number of recovered patients. You can change the dates and the attributes in the space provided below every visualization.

The API used to get the relavent data is by Narrativa. Details on how to use this API is in the link i have mentioned below. https://api.covid19tracking.narrativa.com/ I used D3.js to create the visualizations, JQuery to perform the neccessary DOM Manipulations, Font awesome for the icons, Google fonts for the fonts, Node.js with Express.js for the backend and Heroku to deploy the application.
